## 🛠 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- React
- Next.js
- Tailwind CSS
- Node.js
- Socket.io

## 💻 Projeto

O Talk to Me é um clone do Google Meet onde é possível entrar em uma sala, criar uma nova sala, digitar no chat e ligar a câmera
